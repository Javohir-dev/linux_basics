# Fayllar Bilan Ishlash

## Cat - Fayl Mazmunini Ko'rish

```bash
cat <fayl-nomi>
```

Fayl ichidagi matnni terminalda ko'rsatadi.

**Misol:**

```bash
cat readme.txt
```

---

## Nano - Fayl Tahrirlash

```bash
nano <fayl-nomi>
```

Terminalda matn faylini tahrirlash uchun ochadi.

**Chiqish:**
- `CTRL + X` - chiqish
- `Y` - o'zgarishlarni saqlash
- `N` - saqlamasdan chiqish

---

## Cp - Faylni Nusxalash

```bash
cp <manba-fayl> <maqsad-fayl>
```

Faylni nusxalaydi.

**Misol:**

```bash
cp index.html backup.html
```

---

## Mv - Fayl Ko'chirish yoki Nomini O'zgartirish

```bash
mv <manba-fayl> <maqsad-fayl>
```

Faylni ko'chiradi yoki nomini o'zgartiradi.

**Nomini o'zgartirish misoli:**

```bash
mv oldname.txt newname.txt
```

**Ko'chirish misoli:**

```bash
mv file.txt /home/user/Documents/
```
