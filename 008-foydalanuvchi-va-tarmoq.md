# Foydalanuvchi va Tarmoq Buyruqlari

## Whoami - Joriy Foydalanuvchi

```bash
whoami
```

Joriy foydalanuvchi nomini ko'rsatadi.

---

## Id - Foydalanuvchi Ma'lumotlari

```bash
id
```

Joriy foydalanuvchi haqida batafsil ma'lumot beradi (user ID, group ID va boshqalar).

---

## Passwd - Parolni O'zgartirish

```bash
passwd
```

Foydalanuvchi parolini o'zgartiradi.

---

## Ping - Tarmoq Aloqasini Tekshirish

```bash
ping <manzil>
```

Ko'rsatilgan manzilga ping yuboradi va javob vaqtini o'lchaydi.

**Misol:**

```bash
ping google.com
```

**To'xtatish:** `CTRL + C` tugmasini bosing.

---

## Ifconfig - Tarmoq Interfeyslari

```bash
ifconfig
```

Tarmoq interfeyslarini ko'rsatadi va sozlash imkonini beradi.

**Eslatma:** Ba'zi tizimlarda `ifconfig` o'rniga `ip a` buyrug'i ishlatiladi.

```bash
ip a
```
