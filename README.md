# Yaratish va O'chirish

## Papka Yaratish

### Mkdir

```bash
mkdir <folder-nomi>
```

Yangi papka yaratadi.

**Misol:**

```bash
mkdir my-project
```

---

## Fayl Yaratish

### Touch

```bash
touch <fayl-nomi>
```

Yangi bo'sh fayl yaratadi.

**Misol:**

```bash
touch index.html
```

---

## O'chirish

### Fayl O'chirish - Rm

```bash
rm <fayl-nomi>
```

Ko'rsatilgan faylni o'chiradi.

**Misol:**

```bash
rm test.txt
```

---

### Papka O'chirish - Rm -r

```bash
rm -r <folder-nomi>
```

Ko'rsatilgan papkani va uning ichidagi **barcha fayllarni** o'chiradi.

**Ehtiyot:** Bu buyruq juda xavfli! Barcha ma'lumotlar qaytarilmasdan o'chiriladi.

---

### Bo'sh Papka O'chirish - Rmdir

```bash
rmdir <folder-nomi>
```

**Faqat bo'sh** papkani o'chiradi. Agar ichida biror narsa bo'lsa, xato beradi.

**Misol:**

```bash
rmdir empty-folder
```
