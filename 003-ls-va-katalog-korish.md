# Ls - Fayllar va Papkalarni Ko'rish

## Oddiy Ls

```bash
ls
```

Joriy katalogdagi fayllar va papkalarni ko'rsatadi.

---

## Ls Parametrlari

### -l (List ko'rinishida)

```bash
ls -l
```

Joriy katalogdagi fayllar va papkalarni **list ko'rinishida** ko'rsatadi. Bu ruxsatlar, egalar, o'lchamlar va sanalarni ko'rsatadi.

---

### -a (Yashirin fayllarni ham ko'rish)

```bash
ls -a
```

Yashirin fayllar va folderlarni ham ko'rsatadi. Linuxda yashirin fayllar `.` bilan boshlanadi.

---

### -la (List + Yashirin)

```bash
ls -la
```

Yashirin fayllar va folderlarni **list ko'rinishida** ko'rsatadi.

---

### -lh (Human-readable o'lchamlar)

```bash
ls -lh
```

Fayl o'lchamlarini inson o'qishi mumkin bo'lgan formatda ko'rsatadi (KB, MB, GB).

---

### -R (Recursive)

```bash
ls -R
```

Joriy katalog va uning ichidagi **barcha papkalarni rekursiv ravishda** ko'rsatadi.

---

## Ma'lum Papkani Ko'rish

```bash
ls <folder-nomi>
```

Ko'rsatilgan papkadagi fayllar va papkalarni ko'rsatadi.

**Misol:**

```bash
ls /home
```
