# Cd - Kataloglar Orasida Harakatlanish

## Oddiy Cd

```bash
cd <folder-nomi>
```

Ko'rsatilgan papkaga o'tadi.

**Misol:**

```bash
cd Documents
```

---

## Maxsus Cd Buyruqlari

### .. (Yuqoriga)

```bash
cd ..
```

Bitta yuqoriga papkaga o'tadi.

---

### ~ (Uy papkasi)

```bash
cd ~
```

Uy papkasiga o'tadi (masalan: `/home/username`).

---

### / (Root papka)

```bash
cd /
```

Root (ildiz) papkaga o'tadi.

---

### - (Oldingi papka)

```bash
cd -
```

Oldingi papkaga qaytadi. Bu juda qulay, agar ikki papka orasida tez-tez o'tishingiz kerak bo'lsa.

**Misol:**

```bash
cd /var/log
cd /home/user
cd -  # /var/log ga qaytadi
```
