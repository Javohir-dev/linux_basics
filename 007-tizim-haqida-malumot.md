# Tizim Haqida Ma'lumot Olish

## Uname - Tizim Ma'lumotlari

```bash
uname -a
```

Tizim haqida to'liq ma'lumot beradi (OS nomi, versiyasi, arxitektura va boshqalar).

---

## Top - Jarayonlarni Ko'rish

```bash
top
```

Joriy tizim jarayonlari va resurslar ishlatilishini ko'rsatadi.

**Chiqish:** `Q` tugmasini bosing.

---

## Htop - Yaxshilangan Top

```bash
htop
```

Tizim jarayonlari va resurslar ishlatilishini **interaktiv va rangli** ko'rinishda ko'rsatadi.

### Htop O'rnatish (Debian/Ubuntu)

```bash
sudo apt install htop
```

---

## Df - Disk Bo'sh Joyi

```bash
df -h
```

Disk bo'sh joyi haqida ma'lumot beradi. `-h` parametri human-readable (inson o'qishi mumkin bo'lgan) formatda ko'rsatadi.

---

## Free - Xotira (RAM) Ma'lumoti

```bash
free -h
```

Xotira (RAM) ishlatilishi haqida ma'lumot beradi.

---

## Neofetch - Chiroyli Tizim Ma'lumoti

```bash
neofetch
```

Tizim haqida chiroyli va rangdor ma'lumotlarni ko'rsatadi (OS, kernel, CPU, GPU, RAM va boshqalar).

### Neofetch O'rnatish (Debian/Ubuntu)

```bash
sudo apt install neofetch
```
